# Google 5-Day AI Agents Intensive

> A hands-on AI Agents course created by Google's ML researchers and engineers. Originally held live (November 10–14, 2025), it is now available as a self-paced Kaggle Learn guide. :contentReference[oaicite:0]{index=0}

---

## Overview

This intensive covers the complete lifecycle of modern AI agents:

- 🤖 Agent Fundamentals
- 🔧 Tools & Model Context Protocol (MCP)
- 🧠 Context Engineering & Memory
- 📊 Agent Evaluation & Observability
- 🚀 Production Deployment & Multi-Agent Systems

By the end of the course you'll be able to build, evaluate, and deploy production-ready AI agents. :contentReference[oaicite:1]{index=1}

---

# Course Schedule

| Day | Topic | Focus |
|------|-------|-------|
| 1 | Introduction to Agents | Agent fundamentals & multi-agent systems |
| 2 | Agent Tools & MCP | Tool use, APIs, Model Context Protocol |
| 3 | Context Engineering | Sessions & Long-Term Memory |
| 4 | Agent Quality | Evaluation, Metrics, Observability |
| 5 | Prototype to Production | Deployment & Agent2Agent (A2A) |
